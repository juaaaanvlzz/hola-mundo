# **Ejercicio de hola mundo**
## Hola mundo en python
### Reseña : 
Python es un lenguaje de programación de alto nivel, interpretado y de propósito general, que destaca por su sintaxis limpia y legible (muy similar al inglés), lo que lo hace fácil de aprender y mantener. Su versatilidad, impulsada por una extensa colección de bibliotecas y frameworks, permite su aplicación en una amplia gama de campos, desde el desarrollo web (Django y Flask) y la automatización de sistemas, hasta su dominio absoluto en la ciencia de datos y el aprendizaje automático (Pandas y TensorFlow), consolidándose como una herramienta fundamental tanto para principiantes como para desarrolladores profesionales.
### Codigo:
``` print( "Hola mundo :)" ) ```
